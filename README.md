# juju-docs
Make things easier with JUJU

## Simple go-to commands

### List models
```
juju models
```
### Switch Context
```
juju switch <modelname>
```
### Show all resources in context
```
juju status
```
### Delete model
```
juju destroy-model model-name
```
### Delete model with strorage
```
juju destroy-model model-name --destroy-storage
```
